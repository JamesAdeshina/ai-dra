# AI-DRA

AI-DRA (AI-powered Digital Rehabilitation Assistant) is a web-based rehabilitation support platform designed to assist stroke survivors with guided upper-limb rehabilitation exercises, movement feedback, progress tracking, reminders, and caregiver support.

## Tech Stack

Frontend:
- Next.js
- TypeScript
- Tailwind CSS
- Shadcn UI

Backend:
- Flask
- MediaPipe

Database:
- PostgreSQL
